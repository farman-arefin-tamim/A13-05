
## 1. What is the difference between `var`, `let`, and `const`?

###  `var`

var keyword used for declaring a variable as function level which can be hoisted.

---

###  `let`

let keyword is used for declaring a variable which can be block scope and can not be access out of the block.

---

###  `const`

const keyword used for declaring a variable which cannot be changed or reassign.
